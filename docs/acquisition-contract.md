# Historical: 32×48 medium-tier acquisition contract

**Retired.** This file is a historical pointer only — do not use it for new work.

The authoritative contract for every Party Character and Opponent body entering
the Battlefield is [`body-sprite-contract.md`](body-sprite-contract.md) ([#250](https://github.com/jsbellamy/nightglass/issues/250)).

The frozen medium-tier rules that lived here (exact 32×48 logical grid, grid
recovery, tier safe boxes, and fixed canvas normalizer) are superseded by
per-asset geometry, opaque-bound fit ceilings, and the flexible normalizer path
described in the body contract. The prior full text is preserved in git history
at the commit before #250 merged.
