# PROTOTYPE — Original-IP art direction

Throwaway UI prototype answering:

> What original visual language gives the four Classes, opponents, Stage
> backdrops, and effects a bright chibi fantasy appeal while remaining
> recognizably distinct from Ragnarok Online and readable at the compact
> Battlefield's native scale?

Run from the project root:

```sh
./prototype/art-direction/run.sh
```

Open <http://127.0.0.1:4174/>.

The user selected **Moonberry Guild** as the primary visual language. The
comparison switcher and rejected Skyglass direction have been removed;
Sunsteel survives only as a readability guardrail for bold contours and warm
contrast at native scale.

The retained live sample is exactly 480×112 CSS pixels. Every Party Member occupies an
exact 32×48 SVG canvas displayed at 1×; the enlarged lineup below is an
inspection aid only. The art is rough vector placeholder work intended to test
silhouette, palette, backdrop, opponent, and effect language—not a production
asset source or an AI-generation quality test.
